1. 初回実行時の準備（重要）
本ツールはブラウザ（Chromium）を自動操作してデータを取得します。ツールを起動すると、必要なブラウザが自動的にインストールされます。
2. 動作環境
OS: Windows 10 / 11 (64bit)

インターネット接続: 必須（スクレイピングおよび初回ブラウザ設置に使用）

3. 設定ファイルの準備
実行ファイル（.exe）と同じフォルダに以下のファイルを配置してください。

settings.txt: ログイン情報や取得ページ範囲を記述したファイル。
input.xlsx: 取得対象の企業リストを記載したファイル。

4. フォルダ構成の確認
実行前、フォルダ内が以下のようになっていることを確認してください。

/（任意のフォルダ）
├── SuperDeliveryScraper.exe  (本プログラム)
├── settings.txt              (設定ファイル)
├── input.xlsx                (入力リスト)
└── tmp/log, csv              (ログや一時ファイルが保存されます)

## 使い方

1. **settings.txtにユーザIDとパスワードを書くこと**
   - ユーザIDとパスワードを`settings.txt`ファイルに記入してください。

2. **ページ指定で任意のページから任意のページまで実行可能**
   - スクレイピングを行う際に、開始ページと終了ページを指定できます。

3. **リクエスト毎の待ち時間の幅を変更できる**
   - 待ち時間は環境変数で設定可能ですが、待ち時間が少なすぎるとメンテナンスページが表示され、詳細画面が開かなくなることがあります。3〜4秒間隔が目安です。

4. **特定の企業に絞る機能がある**

5. **ログファイルは7日経過すると自動で削除**
   - 古いログファイルは自動的に削除され、管理が容易です。

6. **処理が途中で止まった場合はtmp配下のcsvにそれまでに取得した詳細情報は残っている**
   - 処理が中断された場合でも、取得したデータは一時ファイルに保存されます。
